The group policy setting for the week this week is a new policy setting for Windows 7/2008 R2. This setting is called "Prevent changing mouse pointers" and can be found under User Configuration > Policies > Administrative Templates > Control Panel > Personalization. This setting is handy if you want to tightly control the user environment such as public access computers in a library or a common shared computer. [![image](https://www.grouppolicy.biz/wp-content/uploads/2010/06/image_thumb36.png)](<https://www.grouppolicy.biz/wp-content/uploads/2010/06/image30.png>) Below you can see how the "pointers" tab is removed when the policy is enabled.

 Be careful however when you apply this setting however as it will locks the use into whatever mouse pointer scheme they had active at the time the policy is applied. P.S.
