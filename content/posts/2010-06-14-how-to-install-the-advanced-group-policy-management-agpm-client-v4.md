### Installing the AGPM Client

It is best you install the Microsoft Advance Group Policy Management Client on any computer in your organisation that has the the Group Policy Management Console (GPMC) installed. This ensures that all changes to group policy are properly managed using the Advanced Group Policy Management program. **Step 1.** Start the Advanced Group Policy Management install program and run the "“ Client install." option. **Step 2.** At Welcome dialog box, click **Next**. [![image](https://www.grouppolicy.biz/wp-content/uploads/2010/01/image_thumb103.png)](<https://www.grouppolicy.biz/wp-content/uploads/2010/01/image108.png>) **Step 3.** Tick **I accept the license terms** and click **Next** [![image](https://www.grouppolicy.biz/wp-content/uploads/2010/01/image_thumb104.png)](<https://www.grouppolicy.biz/wp-content/uploads/2010/01/image109.png>) **Step 4.** Confirm the install patch and click **Next** [![image](https://www.grouppolicy.biz/wp-content/uploads/2010/01/image_thumb105.png)](<https://www.grouppolicy.biz/wp-content/uploads/2010/01/image110.png>) **Step 5.** Type the IP or DNS Name of the AGPM server and click **Next** [![image](https://www.grouppolicy.biz/wp-content/uploads/2010/01/image_thumb106.png)](<https://www.grouppolicy.biz/wp-content/uploads/2010/01/image111.png>) **Step 6.** Leave all the languages selected and click**Next** [![image](https://www.grouppolicy.biz/wp-content/uploads/2010/01/image_thumb107.png)](<https://www.grouppolicy.biz/wp-content/uploads/2010/01/image112.png>) **Step 7.** Click **Install** [![image](https://www.grouppolicy.biz/wp-content/uploads/2010/01/image_thumb108.png)](<https://www.grouppolicy.biz/wp-content/uploads/2010/01/image113.png>) **Step 7a.** Optional "“ Click on the **Details** button to see the components that will be installed.
