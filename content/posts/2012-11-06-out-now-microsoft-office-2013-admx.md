[![Ofc_Orng166_rgb](https://www.grouppolicy.biz/wp-content/uploads/2012/11/OfficelogoOrange_Web_thumb.png)](<https://www.grouppolicy.biz/wp-content/uploads/2012/11/OfficelogoOrange_Web.png>)Microsoft has just release the Office 2013 ADMX/ADML file that allow you to manage the program using Group Policy. You might note that the only the ADMX/ADML (not ADM) files are released for this version of the Office as the program is no longer supported on Windows XP.

[![image](https://www.grouppolicy.biz/wp-content/uploads/2012/11/image_thumb1.png)](<https://www.grouppolicy.biz/wp-content/uploads/2012/11/image1.png>)

Get the ADMX/ADML from: <http://www.microsoft.com/en-au/download/details.aspx?id=35554\>

If you are after the settings spread sheet that details all the policy setting for the Office 2013 still download them from the here <http://www.microsoft.com/en-au/download/details.aspx?id=30341>
